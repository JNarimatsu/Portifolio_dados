Análise de preços de laptop

Projeto de análise exploratória e comparação de modelos de machine learning aplicado ao dataset prices_laptop (Kaggle), que contém informações sobre marcas, configurações e preços de laptops. Foram utilizados os modelos: Regressão Linear, K-Nearest Neighbors (KNN), Support Vector Machine (SVM) e Decision Tree — com e sem ajuste de hiperparâmetros

Previsão de Vendas e Escolha de Loja

Projeto de análise exploratória de quatro conjuntos de dados correspondentes às lojas 01, 02, 03 e 04, pertencentes ao Sr. João, que deseja decidir qual loja deve ser vendida. Os dados incluem informações sobre faturamento, categorias de produtos, produtos vendidos e valores de frete. A análise comparou os resultados de cada loja e utilizou gráficos para facilitar a visualização dos KPIs.

Análise de Diversidade 2022

Análise de dados sobre diversidade na área de tecnologia no ano de 2022. Os dados foram encontrados na pesquisa anual do Datahackers de 2022. Na análise utilizamos Python com as bibliotecas Pandas, Numpy e matplotlib para realizar análise exploratória e identificar a distribuição de profissionais de tecnologia e a diversidade na área. Além da análise realizada em python utilizando as bibliotecas pandas, matplotlib e numpy realizamos as visualizações mais completas dos dados no looker studio.
